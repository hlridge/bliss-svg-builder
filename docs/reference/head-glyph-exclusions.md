# Head Glyph Exclusions

When a word-level indicator (`;;`) is used, or when `;` is used on a predefined multi-character word, the library determines which character receives the indicator (the head glyph). Characters or character sequences in the following list are skipped from the start of the word, so the indicator lands on the first non-excluded character.

See the [Words & Sentences handbook page](/handbook/writing/words-sentences#the-head-glyph-algorithm) for how the algorithm works.
