# Custom Codes

Extend and customize the DSL with your own code systems, aliases, and character definitions.

## Overview

Custom Codes lets you adapt Bliss SVG Builder to your workflow by:
- Creating human-readable aliases (e.g., `LOVE` instead of `B431`)
- Mapping to different ID systems (e.g., BCI-AV-IDs, Blissary IDs)
- Registering custom character definitions
- Bulk-loading character sets
