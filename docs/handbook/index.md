# The Handbook

The [Get Started](/get-started/installation-setup) guide covers the basics — installing the library, rendering your first character, combining characters into words and sentences, and applying simple styling. If you haven't gone through it yet, start there.

The Handbook picks up where Get Started leaves off. Each section goes deeper into a specific area, teaching you the full capabilities of the library.

## What's Covered

### [Writing](/handbook/writing/characters-bcodes)

How to express meaning — compound characters, the indicator system, shape primitives, Latin and Cyrillic text, and custom character definitions.

### [Styling](/handbook/styling/color-stroke)

How to control appearance — colors, stroke widths, fills, backgrounds, accessibility metadata, and SVG output settings.

### [Spacing & Layout](/handbook/spacing-layout/spacing)

How to control spatial arrangement — word and character spacing, kerning, SVG sizing modes, margins, cropping, and element positioning.

### [Coordinate System](/handbook/coordinate-system/understanding-the-grid)

How the grid works — the unit system, coordinate zones, and customizing the grid's visual appearance.

### [DSL Syntax](/handbook/dsl-syntax/syntax-overview)

The complete syntax reference — full grammar specification, the options cascade system, programmatic options via JavaScript, and real-world recipes.

## How to Use the Handbook

The sections build on each other — Writing assumes you know the basics from Get Started, Styling assumes you can write Bliss expressions, and so on. But you can also jump directly to any section if you're looking for something specific.

Each topic is covered in one place. When a page references a concept explained elsewhere, it links to that page rather than re-explaining it.
